## Opgave
Schrijf een programma dat een natuurlijk getal vraagt en vervolgens een rechts uitgelijnde omgekeerde driehoek met het symbool `X` op het scherm weergeeft.

#### Voorbeelden
Geeft de gebruiker `4` in, dan verschijnt er:
```
XXXX
 XXX
  XX
   X
```

Geeft de gebruiker `6` in, dan verschijnt er:
```
XXXXXX
 XXXXX
  XXXX
   XXX
    XX
     X
```
