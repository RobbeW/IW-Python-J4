## Opgave

Gegeven een `lijst` met getallen, schrijf een functie `draai_om(lijst)` die een nieuwe lijst retourneert met de elementen van de `lijst`, maar in omgekeerde volgorde.

Bestudeer onderstaande voorbeelden grondig.

#### Voorbeelden

```python
>>> draai_om([10, 4, 8, 3])
[3, 8, 4, 10]
```

```python
>>> draai_om([2, 2, 2])
[2, 2, 2]
```
